# Lost Mines of Phandelver — Player Notes Repository

This repository serves as a collection of **player-facing notes, lore, and world details** for an ongoing _Lost Mines of Phandelver_ campaign.

It is intended to give players a centralized place to reference information their characters have discovered throughout the adventure, including:

- Locations and regions
- Important NPCs
- Factions and rumors
- Items, artifacts, and clues (where applicable)
- Session summaries (if applicable)

All content here is **spoiler-safe** and limited to what the party has learned in-game. Hidden information, DM notes, and unrevealed plot elements are intentionally excluded.

The goal of this repository is to help maintain continuity, improve immersion, and give players an easy way to stay connected to the world between sessions.
